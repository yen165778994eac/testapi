# API и DevTools тестирование на oort.qahacking.ru

**Цель:** продемонстрировать навыки работы с API (Postman) и инструментами разработчика (DevTools) на тестовом сайте.

**Объект тестирования:** https://oort.qahacking.ru/

## Что сделано

### 1. API-тестирование (Postman)
- Перехвачены и проанализированы сетевые запросы сайта через DevTools (вкладка Network).
- На основе перехваченных запросов создана коллекция в Postman.
- Для ключевых запросов (например, отправка формы, загрузка данных) написаны простые тесты на статус-коды и структуру ответа.
- Коллекция экспортирована и сохранена в папке `postman/`.

### 2. Работа с DevTools
- Проведён анализ всех запросов при загрузке страницы (типы, статусы, размер).
- Проверена консоль (Console) на наличие ошибок и предупреждений.
- Изучена структура DOM и стилей (вкладка Elements).
- Результаты наблюдений описаны в папке `devtools/`, добавлены скриншоты.

## Выводы
Опыт показал, что DevTools — это мощный инструмент для «подкапотного» анализа работы веб-приложений, а Postman позволяет тестировать API независимо от интерфейса, что критично для поиска скрытых дефектов.
